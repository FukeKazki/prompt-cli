# Vision

## Product Name

prompt-cli

# Mission

プロンプト管理を楽にして、再現性のあるAI利用を推進する。

# Vision

prompt-cli は、プロンプトをローカルに保存し簡単に利用、修正ができるようにする。
プロンプトの設計にはfaceted-prompting(https://nrslib.com/faceted-prompting)を利用する。
プロンプトを「Persona」「Policy」「Instruction」「Instruction」「Output Contract」に分解して管理し、結合したものをプロンプトとする設計。
